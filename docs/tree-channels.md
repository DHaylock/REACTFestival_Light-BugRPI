Channels for tree uplighters
============================

3 Channel DMX lights use lightbug parameters:

    dmxchannels: 3
    dmxoffset: 1

Dragons and LEDj Quads use lightbug parameters (and can be mixed):

    dmxchannels: 8
    dmxoffset: 5
